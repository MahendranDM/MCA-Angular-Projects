## 🚀 Projects Included

### 1. Campus360 (`campus360/`)
- **Description**: Web application dashboard built with AngularJS. Includes course study materials and notes in the `notes/` directory.
- **Key Features**: AngularJS app initialization, controller setup (`DashboardController`), and embedded study reference material.

### 2. AngularJS Demo (`angularjs-demo/`)
- **Description**: Demonstrates fundamental AngularJS core concepts.
- **Key Features**: Two-way data binding (`ng-model`), controller state management (`MainController`), styled container layout.

## 🛠️ Getting Started

### Option 1: Direct Browser Launch
Open the `index.html` file inside any project folder (`campus360/index.html` or `angularjs-demo/index.html`) directly in your web browser.

### Option 2: Local Static Server (Recommended)
You can serve the projects using VS Code Live Server extension or `http-server`:

```bash
# Install http-server globally (if not already installed)
npm install -g http-server

# Serve the repository root
http-server .
```
